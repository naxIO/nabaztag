API Info

Command Description Rabbit Values Example Notes
color Set the color of all LEDs V1 0 thru 255 http://nabaztaglives.com/api.php?sn=##:##:##:##:##:##&color=2 (1)
tts Speak a TTS message V2 255 characters http://nabaztaglives.com/api.php?sn=##:##:##:##:##:##&tts=hello (2)
play Play an MP3 V2 240 character URL http://nabaztaglives.com/api.php?sn=##:##:##:##:##:##&play=nabaztaglives.com/avionics.mp3 (3)
stream Stream an MP3 V2 240 character URL http://nabaztaglives.com/api.php?sn=##:##:##:##:##:##&stream=nabaztaglives.com/avionics.mp3 (3)

(1) ##:##:##:##:##:## is the serial number of your rabbit
(2) This service uses Google TTS
(3) The V2 suffers from buffering issues which makes the sound "crack" or "break up". You can choose any http based URL you want that returns an MP3 (do not include http://). The payload is between the URL you choose and your rabbit; it does not come back to this server for any processing.
